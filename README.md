# Bamazon 1.0.0

## Overview
	* Bamazon is an Amazon-like storefront program using Node.js and Mysql database.

## NPM Packages 
* mysql
```bash
npm install mysql
```
* inquirer
```bash
npm install inquirer
```
* cli-table3
```bash
npm install cli-table3
```
* color
```bash
npm install colors
```
* emoji
```bash
npm install node-emoji
```

## LIRI Commands
* concert-this
* spotify-this-song
* movie-this
* do-what-it-says
